# ReleaseCheck Web

The web interface should make platform availability and matching evidence visible at a glance.

Run locally:

```bash
bun run dev:web
```

Or run the API and web UI together from the repository root:

```bash
bun run dev:demo
```

Open `http://localhost:3001`.

Phase 2 demo surfaces:

- Search
- Release detail
- Platform availability
- Match evidence
- Index status
- Unknown states
- Version and same-title ambiguity
