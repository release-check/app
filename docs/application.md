# Competition Application

## 프로젝트명

ReleaseCheck

## 개발 목적

ReleaseCheck는 국내외 음악 플랫폼에 흩어진 음원과 앨범 정보를 한 번에 확인하는 오픈소스 검색 API를 개발한다. 단순히 검색 결과를 나열하지 않고, 아티스트명, 곡명, 발매일, 재생시간, 외부 식별자 등을 비교해 동일한 음원 또는 앨범 여부를 판단하고 신뢰도 점수와 매칭 근거를 함께 제공한다.

## 프로젝트 소개

ReleaseCheck는 Spotify, Apple Music, YouTube 같은 글로벌 플랫폼과 Melon 같은 국내 음악 서비스, MusicBrainz, Discogs 등 공개 메타데이터를 함께 다루는 오픈소스 시스템이다. 초기 버전은 공개 메타데이터 데이터베이스와 공식 API 또는 공개 데이터 접근이 가능한 음악 플랫폼부터 우선 연동한다. 검색 결과는 제공 여부, 링크, 신뢰도 점수, 매칭 근거로 정리해 API와 CLI로 제공하고, 웹 인터페이스에서는 플랫폼별 결과를 한 화면에서 확인한다.

## 기대 효과

같은 음악을 찾으려고 플랫폼을 계속 옮겨 다니는 시간을 줄인다. 독립 아티스트, 소규모 레이블, 언더그라운드 음악은 국내외 플랫폼마다 등록 정보가 다르거나 일부 플랫폼에만 올라오는 일이 많아 찾기가 어렵다. ReleaseCheck는 이런 음악의 제공 여부를 공통 API로 조회할 수 있게 해 리스너, 큐레이터, 음악 블로거, 아카이브 운영자가 음악을 더 쉽게 추적하도록 돕는다.

개발자는 플랫폼별 API를 하나씩 붙이지 않아도 통합 음악 검색 기능을 구현한다. 매칭 근거를 함께 보여주기 때문에 잘못된 결과를 검토하기 쉽고, 음악 데이터 연구나 추천 밖 음악을 다루는 서비스에서도 재사용된다. 오픈소스 구조로 공개해 새로운 플랫폼 연동과 매칭 규칙 개선도 외부 기여를 받아 확장한다. 장기적으로는 국내외 플랫폼별 제공 여부와 메타데이터를 연결한 공개 음악 메타데이터 그래프를 구축하는 것을 목표로 한다.
